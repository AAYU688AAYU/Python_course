## 📘 Learning Python Repository

This repository contains structured learning materials and practice exercises for mastering Python programming. It covers fundamental to intermediate concepts through hands-on examples and chapter-wise code files.

### 📚 Topics Covered

* Variables and Data Types
* Conditional Statements and Loops
* Functions and Modular Programming
* Data Structures (Lists, Tuples, Dictionaries, Sets)
* String Manipulation
* File Handling
* Error and Exception Handling
* Object-Oriented Programming (OOP)
* Basic use of Python libraries and APIs

### 🎯 Purpose

The goal of this repository is to build a strong foundation in Python and develop problem-solving skills through practical implementation.

### ⚙️ How to Run

1. Clone the repository
2. Navigate to the desired chapter folder
3. Run Python files using:

   ```
   python filename.py
   ```

### 🚀 Why Python?

Python is simple, readable, and powerful, making it ideal for beginners as well as for advanced applications like AI, Machine Learning, Web Development, and Automation.
